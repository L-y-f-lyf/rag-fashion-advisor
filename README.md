# RAG Fashion Advisor — AI 时尚穿搭顾问

基于检索增强生成（RAG）+ 大模型打造的智能穿搭推荐系统。
根据用户场景、风格、身材，提供个性化时尚建议。

## 功能
- 智能穿搭推荐
- 场景化穿搭生成（约会、职场、休闲等）
- 基于知识库的专业时尚问答
- 自然语言多轮对话

## 技术栈
- Python
- LangChain / RAG
- LLM 大模型
- Streamlit（前端展示）
- GitHub 托管 & 一键部署

## 如何运行
```bash
pip install -r requirements.txt
streamlit run app.py
