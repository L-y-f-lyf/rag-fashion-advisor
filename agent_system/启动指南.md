# 👔 穿搭智能顾问 Agent 系统 - 启动指南

## 📋 快速启动步骤

### 步骤1：进入项目目录
```bash
cd agent_system
```

### 步骤2：安装依赖
```bash
pip install -r requirements.txt
```

### 步骤3：配置 API 密钥（重要！）

#### 方式1：使用 .env 文件（推荐）
```bash
# 复制示例文件
cp .env.example .env

# 编辑 .env 文件，填入你的通义千问 API 密钥
# 将 "你的API密钥" 替换为真实密钥，例如：
# DASHSCOPE_API_KEY=sk-xxxxxxxxxxxx
```

#### 方式2：直接修改 config.py
打开 `config.py` 文件，找到第8行左右：
```python
DASHSCOPE_API_KEY = "sk-xxxxxxxxxxxx"  # 填入你的真实API密钥
```

#### 方式3：启动后在界面中输入
- 启动系统后，在页面顶部的 API Key 输入框中填入

### 步骤4：启动系统

#### 方式1：使用启动脚本（推荐）
```bash
python start.py
```

#### 方式2：直接运行 main.py
```bash
python main.py
```

### 步骤5：访问系统

- **前端页面：** http://127.0.0.1:8000
- **API 文档：** http://127.0.0.1:8000/docs
- **健康检查：** http://127.0.0.1:8000/health

## ⚠️ 常见问题

### Q1: 提示找不到 chroma_db？
**A:** 检查向量数据库路径，确保 `chroma_db` 目录存在于 `agent_system` 文件夹中

### Q2: API 调用失败？
**A:**
- 检查 API 密钥是否正确
- 检查网络是否正常
- 检查 API 密钥是否有额度

### Q3: 依赖安装失败？
**A:** 确保使用 Python 3.8+，尝试更新 pip：
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### Q4: 端口 8000 被占用？
**A:** 在 `config.py` 中修改端口号：
```python
PORT = 8001  # 改为其他端口
```

## 📝 API 接口说明

### 1. Agent 聊天接口
```bash
POST /agent/chat
```
请求体：
```json
{
  "message": "我是个女生，梨形身材，要去约会，想要轻熟风格",
  "api_key": "可选，如已配置可省略"
}
```

### 2. RAG 问答接口
```bash
POST /rag/chat
```
请求体：
```json
{
  "query": "春天适合穿什么颜色的衣服",
  "api_key": "可选"
}
```

### 3. 穿搭方案生成接口
```bash
POST /outfit/generate
```
请求体：
```json
{
  "gender": "女",
  "scene": "约会",
  "body_type": "梨形",
  "style": "轻熟",
  "additional_request": "希望显瘦一点",
  "api_key": "可选"
}
```
