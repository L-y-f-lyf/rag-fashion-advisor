# 安装说明 - 解决 langchain_core.pydantic_v1 错误

## 问题
当前环境缺少必要的依赖包。

## 解决方法

### 方法 1：在 PyCharm 终端中运行（推荐）

打开 PyCharm 底部的 "Terminal"（终端）标签，然后依次运行以下命令：

```bash
cd agent_system
```

```bash
pip install -r requirements.txt
```

### 方法 2：使用 PyCharm 的 Python 包管理器

1. 在 PyCharm 中，打开 `File` → `Settings` (Windows) 或 `PyCharm` → `Preferences` (Mac)
2. 找到 `Project: PythonProject1` → `Python Interpreter`
3. 点击 `+` 号添加包
4. 搜索并安装以下包（注意版本号）：
   - `pydantic` → 版本 `1.10.15`
   - `langchain` → 版本 `0.1.20`
   - `langchain-core` → 版本 `0.1.52`
   - `langchain-community` → 版本 `0.0.38`
   - `langchain-chroma` → 版本 `0.1.2`
   - `fastapi` → 版本 `0.109.0`
   - `uvicorn` → 版本 `0.27.0`
   - `dashscope` → 版本 `1.14.1`
   - `chromadb` → 版本 `0.4.24`
   - `python-dotenv` → 版本 `1.0.0`
   - `pydantic-settings` → 版本 `2.1.0`

## 验证安装

安装完成后，在终端中运行：

```bash
python -c "from langchain_community.chat_models import ChatTongyi; print('✅ 安装成功！')"
```

如果看到 "✅ 安装成功！"，说明问题已解决。

## 启动系统

安装成功后，运行：

```bash
python start.py
```
